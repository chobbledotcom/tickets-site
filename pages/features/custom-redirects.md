      ## Post-Booking Experience

      Configure a custom redirect URL so that after completing their
      booking, attendees are sent to your own thank-you page, a survey, a
      resource page, or any URL you choose. If no redirect is set, they
      see the default confirmation page.

      ## Use Cases

      - **Thank-you page** on your main website with next steps
      - **Survey** to collect feedback immediately after booking
      - **Resource page** with preparation materials or directions
      - **Upsell page** offering merchandise or related events

      ## Simple Setup

      Set the redirect URL in your event or global settings and it applies
      immediately. No code changes or template editing required.
